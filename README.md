# Instagram UI Clone

A very basic Instagram UI clone built using **React** and **CSS**.  
This project focuses on recreating the user interface and basic page flow.

---

## Features

- Login / Signup page
- Sidebar navigation
- Profile page
- Responsive layout
- User data stored after signup
- Dark theme inspired by Instagram
- Clean component-based structure

---

## Tech Stack

- React
- JavaScript
- CSS
